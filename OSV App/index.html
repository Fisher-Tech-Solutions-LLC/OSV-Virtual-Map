Old Sturbridge Village Virtual Map

About the Virtual Map Project

The virtual map project was developed as a way to provide the visitors of Old Sturbridge Village a method of finding their way around the grounds without needing anything other than the use of their phone. The map uses Google Maps Directions API to provide users with accurate directions to various locations integral to the OSV visitor experience, including each major historical exhibit and house as well as neccessities such as bathrooms, food and gift shop locations

Want to support this Project?

The OSV Virtual Map is a free for the visitors of OSV and always will be due to the fact that the project is open source. If you would like to help the project by reporting bugs, updating geo-coordinate data, or providing feedback, you can do so by visiting the repository on Github here: OSV Virtual Map Project
